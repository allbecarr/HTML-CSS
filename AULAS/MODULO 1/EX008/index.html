Principais formatações

Negrito / Destaque

Nesta frase temos um tesrmo em negrito usando a tag '<b>' (não semântica).

Nesta frase temos um tesrmo em destaque usando a tag '<strong>' (semântica).

Itálico / Ênfase

Nesta frase temos um tesrmo em itálico usando a tag '<i>' (não semântica).

Nesta frase temos um tesrmo em ênfase usando a tag '<em>' (semântica).

Texto Mrcado

Podemos criar um texto marcado usando a tag <mark>.

Aplicando estilo ao texto marcado usando o parametro style da CSS para mudar a cor do marcador.

Texto grande e texto pequeno

Estamos criando um texto grande e outro texto pequeno usando as tags <big> e <small> (não semântica).

Texto Excluido ou Deletado

Podemos marcar um texto como excluido para indicar que ele pode ser lido, porém deverá ser desconsiderado usando a tag <del>. (semântica)

Texto Incluído

Podemos marcar um texto como inserido para dar ênfase ou indicar que o mesmo foi adicionado posteiormente usando a tag <ins>. (semântica)

Existe também o sublinhado usando a tag <u>. (não semântica)

Texto Sobrescrito

Para inserir textos sobrescritos com x20+3, usamos a tag <sup>.

Texto subscrito

Para inserir textos sobscritos com H2o, usamos a tag <sub>.
